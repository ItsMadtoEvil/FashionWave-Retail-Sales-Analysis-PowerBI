# 👗 FashionWave Retail Sales Analysis | Power BI Dashboard

## 📌 Project Overview

FashionWave is a Business Intelligence project developed using Power BI to analyze retail sales data across 50 stores in India.

The objective was to identify sales trends, customer behavior, product performance, discount impact, and store performance to help management make data-driven business decisions.

---

## 🎯 Business Objectives

- Increase weekend sales by 15%
- Identify high-margin product categories
- Analyze discount impact on profitability
- Evaluate store performance
- Understand customer segments
- Improve marketing strategy

## 📂 Dataset

The dataset contains over **30,000 retail transactions** across **50 stores**.

Main columns include:

- Invoice Number
- Date
- Store ID
- Product ID
- Category
- Quantity
- Unit Price
- Discount
- Cost Per Unit
- Customer Type
- Payment Mode

## 📈 KPIs

- Total Revenue: ₹119.46M

- Profit Margin: 28.46%
- Transactions: 30K
- Units Sold: 75K
- Average Order Value: ₹3.98K

---

## 💡 Key Business Insights

- Accessories generated the highest profit margin.
- Clothing category requires cost optimization.
- Weekend sales contribute only 29% of total revenue.
- Heavy discounts significantly reduce profitability.
- Top-performing stores outperform bottom stores due to better footfall and marketing.
